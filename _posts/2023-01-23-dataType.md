 ## JAVA 자료형



```java
public class datatype {
    public static void main(String[] arg){
        //참, 거짓을 저장할 수 있는 자료형
        boolean a = true;  
        boolean a = false; 
      
        //정수를 저장할 수 있는 자료형
        //byte, short, int, long
        byte a = 1;
        short a = 1;
        int a = 1;
        long a = 1;
        
        //실수를 저장할 수 있는 자료형
        //float, double
        double a = 1.1;
        float a = 1.1;
        
        //문자를 저장하는 자료형 char
        //문자 : 한 글자, 반드시 홀따옴표로 감싸야 함.
        //문자열 : 하나 이상의 글자, 반드시 쌍따옴표로 감싸야 함.
        char c = 'A';
        
        //문자열을 저장하는 자료형
        //쌍따옴표로 감싸야 함
        String a = "java"
          
    }
}
```

## 타입변환

실수를 적으면(1.1) 자바는 기본적으로 double형으로 인식

```java
public class datatype{
    public static void main(String[] args) {
        //자동 타입 변환 - promotion
        int x = 10;
        double x2 = x1;     //x2 = 10.0
      
        //강제 타입 변환 - casting
        double y1 = 10.5;
        int y2 = (int)y1;   //y2 = 10
    }
}

```
